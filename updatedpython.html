Introduction to Python

Python is a versatile, high-level programming language known for its simplicity and readability. It is widely used in web development, data science, machine learning, automation, and more.

Getting Started

  1. Install Python: Download from python.org.
  2. Set up an IDE: Use PyCharm, VS Code, or Jupyter Notebook.
  3. Write your first program: 
    
# First Python Program
print("Hello, World!")

Python Basics

  • Data Types: int, float, str, list, tuple, dict, set
  • Control Structures: if, for, while
  • Functions: 
    
def greet(name):
    return f"Hello, {name}!"
  • File Handling: 
    
with open("example.txt", "w") as file:
    file.write("This is a Python file!")

Practical Examples

1. Fibonacci Sequence


def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        print(a, end=" ")
        a, b = b, a + b

fibonacci(10)

2. Sorting a List


numbers = [5, 2, 9, 1, 5, 6]
sorted_numbers = sorted(numbers)
print("Sorted List:", sorted_numbers)

3. Web Scraping


import requests
from bs4 import BeautifulSoup

response = requests.get("https://example.com")
soup = BeautifulSoup(response.text, "html.parser")
print(soup.title.text)

Working with APIs

Python makes it simple to interact with APIs using libraries like requests and httpx:


import requests

response = requests.get("https://api.example.com/data")
if response.status_code == 200:
    data = response.json()
    print(data)

Python Interview Questions

  1. What are Python's key features?

    Easy syntax, interpreted, dynamically typed, extensive libraries, and platform-independent.

  2. What are Python's data types?

    Basic: int, float, str
    Collections: list, tuple, set, dict

  3. How do you handle exceptions in Python? 
    
try:
    x = 1 / 0
except ZeroDivisionError as e:
    print("Error:", e)
  4. What is the difference between a list and a tuple?

    Lists are mutable, tuples are immutable.

  5. Write a Python program to reverse a string. 
    
string = "Python"
print(string[::-1])
