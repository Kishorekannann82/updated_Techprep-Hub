Excel Data Analytics Course: Essential Formulas & Shortcuts

Welcome to the Excel Data Analytics Course. This guide covers essential Excel formulas for data analysis and useful keyboard shortcuts to help you navigate and perform tasks more efficiently in Excel.

1. Basic Excel Formulas

  • SUM: Adds up a series of numbers.
    • =SUM(A1:A10)
  • AVERAGE: Returns the average of a range of numbers.
    • =AVERAGE(A1:A10)
  • COUNT: Counts the number of cells that contain numbers.
    • =COUNT(A1:A10)
  • COUNTA: Counts the number of non-empty cells.
    • =COUNTA(A1:A10)
  • MIN: Returns the smallest value in a range of numbers.
    • =MIN(A1:A10)
  • MAX: Returns the largest value in a range of numbers.
    • =MAX(A1:A10)
  • IF: Performs a logical test and returns one value if TRUE and another if FALSE.
    • =IF(A1 > 10, "Yes", "No")

2. Lookup & Reference Formulas

  • VLOOKUP: Looks for a value in the first column of a table and returns a value in the same row from another column.
    • =VLOOKUP(A1, B1:C10, 2, FALSE)
  • HLOOKUP: Similar to VLOOKUP but searches for data in rows instead of columns.
    • =HLOOKUP(A1, A2:D10, 2, FALSE)
  • INDEX: Returns the value of a cell in a given row and column from a specified range.
    • =INDEX(A1:C10, 2, 3)
  • MATCH: Returns the relative position of an item in a range that matches a specified value.
    • =MATCH("Product A", A1:A10, 0)

3. Logical Formulas

  • AND: Returns TRUE if all conditions are TRUE.
    • =AND(A1 > 10, B1 < 5)
  • OR: Returns TRUE if any condition is TRUE.
    • =OR(A1 > 10, B1 < 5)
  • NOT: Reverses the logical value (TRUE becomes FALSE and vice versa).
    • =NOT(A1 > 10)

4. Date & Time Formulas

  • TODAY: Returns the current date.
    • =TODAY()
  • NOW: Returns the current date and time.
    • =NOW()
  • DATE: Returns a date based on a specified year, month, and day.
    • =DATE(2023, 12, 31)
  • DAYS: Calculates the number of days between two dates.
    • =DAYS(A1, A2)

5. Statistical Formulas

  • STDEV: Calculates the standard deviation of a set of values.
    • =STDEV(A1:A10)
  • VAR: Calculates the variance of a set of numbers.
    • =VAR(A1:A10)
  • CORREL: Returns the correlation coefficient between two data sets.
    • =CORREL(A1:A10, B1:B10)

6. Financial Formulas

  • PMT: Calculates the payment for a loan based on constant payments and a constant interest rate.
    • =PMT(interest_rate, number_of_periods, loan_amount)
  • FV: Calculates the future value of an investment based on constant payments and a constant interest rate.
    • =FV(rate, nper, pmt, [pv], [type])

Useful Excel Shortcuts for Efficiency

  • Ctrl + C: Copy selected cells.
  • Ctrl + V: Paste copied cells.
  • Ctrl + Z: Undo the previous action.
  • Ctrl + Shift + L: Toggle filters on or off.
  • Ctrl + Arrow Key: Navigate to the edge of the data.
  • Alt + E, S, V: Paste special (useful for pasting values or formatting).
  • Ctrl + T: Convert a range of data into a table.
  • F2: Edit the selected cell.
