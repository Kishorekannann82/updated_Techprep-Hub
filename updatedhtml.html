Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the basic structure of a website, which is enhanced and modified by CSS and JavaScript. HTML is easy to learn and widely used by developers worldwide.

A Brief History of HTML

HTML was created by Tim Berners-Lee in 1991 as a way to structure content on the World Wide Web. Over the years, it has evolved significantly:

  • HTML 1.0: The first version, focused on simple web pages.
  • HTML 4.01: Introduced CSS integration and improved functionality.
  • HTML5: Released in 2014, it brought modern features like multimedia support, semantic elements, and APIs for dynamic content.

Basic HTML Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to HTML</h1>
    <p>HTML is easy to learn!</p>
  </body>
</html>

HTML with CSS

CSS (Cascading Style Sheets) is used to style HTML content. You can add CSS in three ways:

  • Inline CSS: Directly within an HTML element using the style attribute.
  • Internal CSS: Inside a <style> tag in the <head>.
  • External CSS: Linked via a <link> tag to an external stylesheet file.

Example of an external CSS file:


/* styles.css */
body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

JavaScript in HTML

JavaScript is a scripting language that brings interactivity to web pages. You can add JavaScript:

  • Inline: Using the onclick or onchange attributes.
  • Internal: Within a <script> tag in the <head> or <body>.
  • External: Linking to a separate .js file with the <script> tag.

Example of adding a JavaScript alert:


<button onclick="alert('Hello, World!')">Click Me</button>

What's New in HTML5?

  • Semantic tags like <article>, <section>, and <aside> for better structure.
  • Built-in support for audio and video with <audio> and <video> tags.
  • Canvas for drawing graphics with JavaScript using the <canvas> tag.
  • Geolocation API for accessing the user's location.
  • Web Storage API for local data storage using localStorage and sessionStorage.

Pro Tips

  • Keep your HTML code clean and well-structured for better readability.
  • Use semantic tags for better SEO and accessibility.
  • Separate content (HTML), styling (CSS), and behavior (JavaScript) for modular development.
  • Always test your web pages on multiple browsers for compatibility.
  • Learn the basics of browser developer tools to debug your code efficiently.
  • Use version control systems like Git to manage your web development projects.
